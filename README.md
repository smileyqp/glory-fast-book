# 项目介绍

Glory-Fast快速开发平台。致力于打造能快速应用于企业开发的快速开发平台。减少基础功能开发，更专注于业务逻辑。

> 后端源码下载：<https://github.com/RhineDream/glory-fast>
>
> 前端源码下载：<https://github.com/smileyqp/glory-fast-web>

#### 技术架构

```shell
前端技术： Ant-design-react + React + Webpack 
```
